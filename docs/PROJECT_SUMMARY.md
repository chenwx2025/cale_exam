## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 初始化数据库
```bash
npm run db:push
npm run db:seed  # 可选：填充示例数据
```

### 3. 启动开发服务器
```bash
npm run dev
```

访问：http://localhost:3001

### 4. 访问管理后台
访问：http://localhost:3001/admin

**默认密码：** `admin123`
