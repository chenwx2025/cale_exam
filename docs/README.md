## 🚀 快速链接

### 新手入门
1. 阅读 [快速开始指南](QUICK_START.md)
2. 运行 `npm install`
3. 运行 `npm run db:push && npm run db:seed`
4. 运行 `npm run dev`

### 开发者
1. 查看 [项目技术概览](PROJECT_OVERVIEW.md)
2. 了解数据库架构
3. 熟悉 API 路由
4. 开始开发新功能

### 部署上线
1. 阅读 [安全性说明](SECURITY.md)
2. 检查生产环境清单
3. 配置环境变量
4. 部署到生产环境
